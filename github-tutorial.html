Overview

Git is a version control system which originated as a way to help teams of developers work collaboratively on code without going insane. It tracks changes to a set of files called a repository.

GitHub provides a free home for these repositories on the internet and has evolved into a neat community of not just software people, but data scientists and other nerds who like to contribute to open-source projects. Not only can you store your files for free (under a certain size), but you can also host a website (like this one) and track progress on a team project.
